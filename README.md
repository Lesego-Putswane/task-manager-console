Task Manager Console App

Project Overview

This is a simple console-based Task Manager application built in Python. It allows users to manage their daily tasks by adding new tasks, viewing existing tasks, marking tasks as done, and deleting completed tasks. This project is designed for practicing Python programming, GitHub workflow, and basic backend logic.

Features

Add new tasks

View tasks that are still pending

Mark tasks as done

View completed tasks

Delete completed tasks


How to Run

1. Clone the repository to your local machine.


2. Navigate to the project folder in your terminal.


3. Run the Python file:

python task_manager.py


4. Follow the on-screen menu to interact with the application.



Usage Instructions

Add Task: Select the option to add a task and type the task description.

View Tasks: Select the option to view all pending tasks.

Mark Task as Done: Choose the task number to mark it as completed.

View Completed Tasks: Displays all tasks marked as done.

Delete Completed Task: Select completed tasks to remove them from the list.

Exit: Close the application when finished.


Notes

Tasks are stored temporarily in memory and will be lost when the program exits.

Input validation is included to prevent crashes due to invalid entries.
